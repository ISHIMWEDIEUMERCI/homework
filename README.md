# E-Trade Login System

## 📌 Description
A simple and modern login system built using HTML, CSS, and JavaScript.  
It includes multiple screens such as login, password reset, and a welcome page with smooth animations.

## 📁 Project Structure
- index.html → Main interface (login system)
- css/style.css → Styling (design, layout, animations)
- js/script.js → Functionality (interactions, navigation)

## 🚀 Features
- Login form interface
- Password show/hide toggle
- Password reset screen
- Smooth page transitions
- Welcome screen after login

## 🎯 Purpose
This project is built to practice frontend development and GitHub project organization skills.
